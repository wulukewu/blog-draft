Setup

  1. 根據email創建一個新的SSH金鑰 ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

  2. 開啟ssh-agent，將私鑰加入到ssh-agent

  3. 上傳公鑰，並到GitHub上上傳SSH key cat ~/.ssh/id_rsa.pub GitHub Keys Settings

  4. ssh連接GitHub SSH ssh -T git@github.com
